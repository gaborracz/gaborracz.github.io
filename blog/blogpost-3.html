How to Create a Simple "Hello, World!" Program in C++

If you’re just starting out with C++, one of the best ways to get comfortable is by writing the classic “Hello, World!” program. This simple exercise introduces you to the basic structure of a C++ program and how to output text to the screen. Here’s a step-by-step guide:

1. Open Your Code Editor or IDE

You can use any text editor (like Notepad, VS Code, or Sublime Text), but for a smoother experience, a C++-friendly IDE such as Code::Blocks, Visual Studio, or CLion is recommended.

2. Write the Code

Type the following code into a new file and save it as hello.cpp: 

#include <iostream>

int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}

What does this code do?

  • #include <iostream> tells the compiler to include the standard input/output stream library, which is necessary for outputting text.
  • int main() defines the main function, which is the entry point of every C++ program.
  • std::cout << "Hello, World!" << std::endl; prints “Hello, World!” to the screen, followed by a newline.
  • return 0; signals that the program ended successfully.

3. Compile the Program

To turn your code into an executable program, you need to compile it. If you’re using an IDE, just click the “Build” or “Run” button. If you’re using the command line, navigate to the folder containing hello.cpp and run: 

g++ hello.cpp -o hello

This creates an executable named hello (or hello.exe on Windows).

4. Run the Program

On the command line, run: 

./hello

(or hello.exe on Windows)

You should see: 

Hello, World!
Conclusion

That’s it! You’ve just written, compiled, and run your first C++ program. This “Hello, World!” example demonstrates the basic structure of a C++ application and how to use standard output. From here, you can start experimenting with more complex features and build your programming skills further. Happy coding!
